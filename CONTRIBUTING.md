# Contributing to the repository

This repository is a collection of templates for [OwlGrid resources](https://owlgrid.com). If you want to contribute, please, follow the next steps:
- Fork the repository
- Create a new branch with the name of the template you want to contribute
- Add the template to the matching resource folder
- Check that the template is valid by running the following command:
```bash
TODO (ex: owlgrid validate <template>)
```
- Check that the template doesn't contain any sensitive information (passwords, tokens, etc.)
- Commit your changes
- Create a pull request on the `main` branch of this repository
- Wait that the pull request is reviewed and merged
- Thank you for your contribution!
